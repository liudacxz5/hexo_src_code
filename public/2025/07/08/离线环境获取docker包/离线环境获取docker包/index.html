离线环境获取docker包

发表于2025-07-08|更新于2025-07-08|开发心得
|浏览量:

在联网机器下载镜像

docker pull mysql:8.0.36

保存镜像为离线包

docker save -o mysql8.tar mysql:8.0.36

传输文件到离线环境后加载镜像

docker load -i mysql8.tar
